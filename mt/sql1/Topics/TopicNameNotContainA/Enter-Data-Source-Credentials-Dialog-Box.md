# Enter Data Source Credentials Dialog Box
  Use the **Enter Data Source Credentials** dialog box to provide your [!INCLUDE[ssNoVersion](../../Token/Other/ssNoVersion_md.md)] credentials so that you can retrieve data from the data source.  
  
## Options  
 **User name**  
 Type the user name to log on to the data source.  
  
 **Password**  
 Type the password to log on to the data source.
