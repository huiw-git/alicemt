# Proxy Account Properties - New Proxy Account (Principals Tab)
Use this page to view or change the principals that can use a [!INCLUDE[msCoName](../content/includes/msCoName_md.md)] [!INCLUDE[ssNoVersion](../content/includes/ssNoVersion_md.md)] Agent proxy account in job steps.  
  
## Options  
**Proxy account principals**  
Lists the principals that can use this proxy account.  
  
**Add**  
Adds a principal to the list.  
  
**Remove**  
Removes the selected principal from the list.
