# getBinaryStream Method (SQLServerCallableStatement)
  Retrieves the value of the designated parameter as a binary stream of uninterrupted bytes.  
  
## Overload List  
  
|Name|Description|  
|----------|-----------------|  
|[getBinaryStream &#40;int&#41;](../content/getBinaryStream--int-.md)|Retrieves the value of the designated parameter as a binary stream of uninterrupted bytes given the parameter index.|  
|[getBinaryStream &#40;java.lang.String&#41;](../content/getBinaryStream--java.lang.String-.md)|Retrieves the value of the designated parameter as a binary stream of uninterrupted bytes given the parameter name.|
