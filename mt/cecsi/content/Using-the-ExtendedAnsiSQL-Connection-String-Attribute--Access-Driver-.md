  <introduction>
    <para>Jet 4.0 has several new features that are not enabled by default in the ODBC Desktop Database Drivers version 4.0. To take advantage of these new features, an application will have to set a new connection attribute, add a new attribute to the connection string, or set the attribute in a DSN. The attribute is called ExtendedAnsiSQL and pertains only to the Microsoft ODBC Desktop Database Drivers 4.0 and later. </para>
