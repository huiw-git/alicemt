# getShort Method (SQLServerResultSet)
  Retrieves the value of the designated column in the current row of this [SQLServerResultSet](../content/SQLServerResultSet-Class.md) object as a **short** in the Java programming language.  
  
## Overload List  
  
|Name|Description|  
|----------|-----------------|  
|[getShort (int)](../content/getShort-Method--int---SQLServerResultSet-.md)|Retrieves the value of the designated column index in the current row of this [SQLServerResultSet](../content/SQLServerResultSet-Class.md) object as a **short** in the Java programming language.|  
|[getShort (java.lang.String)](../content/getShort-Method--java.lang.String---SQLServerResultSet-.md)|Retrieves the value of the designated column name in the current row of this [SQLServerResultSet](../content/SQLServerResultSet-Class.md) object as a **short** in the Java programming language.|
