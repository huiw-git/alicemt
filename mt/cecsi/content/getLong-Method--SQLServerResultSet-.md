# getLong Method (SQLServerResultSet)
  Retrieves the value of the designated column in the current row of this [SQLServerResultSet](../content/SQLServerResultSet-Class.md) object as a **long** in the Java programming language.  
  
## Overload List  
  
|Name|Description|  
|----------|-----------------|  
|[getLong (int)](../content/getLong-Method--int---SQLServerResultSet-.md)|Retrieves the value of the designated column index in the current row of this [SQLServerResultSet](../content/SQLServerResultSet-Class.md) object as a **long** in the Java programming language.|  
|[getLong (java.lang.String)](../content/getLong-Method--java.lang.String---SQLServerResultSet-.md)|Retrieves the value of the designated column name in the current row of this [SQLServerResultSet](../content/SQLServerResultSet-Class.md) object as a **long** in the Java programming language.|
