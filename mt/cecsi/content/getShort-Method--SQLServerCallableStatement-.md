# getShort Method (SQLServerCallableStatement)
  Retrieves the value of the designated parameter as a **short** in the Java programming language.  
  
## Overload List  
  
|Name|Description|  
|----------|-----------------|  
|[getShort \(int\)](../content/getShort-Method--int-.md)|Retrieves the value of the designated parameter as a **short** in the Java programming language given the parameter index.|  
|[getShort \(java.lang.String\)](../content/getShort-Method--java.lang.String-.md)|Retrieves the value of the designated parameter as a **short** in the Java programming language given the parameter name.|
