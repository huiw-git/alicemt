# SQLServerClob Class
  Represents a character large binary object (CLOB).  
  
 **Package:** com.microsoft.sqlserver.jdbc  
  
 **Extends:** java.lang.Object  
  
 **Implements:** java.sql.Clob, java.io.Serializable  
  
## Syntax  
  
```  
  
public class SQLServerClob  
```  
  
## Remarks  
 A CLOB is stored in [!INCLUDE[ssNoVersion](../content/includes/ssNoVersion_md.md)] as a TEXT or NTEXT data type.
