# getContents Method (SQLServerResource)
  Retrieves error string information. This method is intended for internal use only.  
  
## Syntax  
  
```  
  
protected java.lang.Object[][] getContents()  
```  
  
## Return Value  
 A multi-dimensional array of **Object** values.
