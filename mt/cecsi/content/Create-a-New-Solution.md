# Create a New Solution
When you create a project, [!INCLUDE[ssManStudioFull](../content/includes/ssManStudioFull_md.md)] creates a solution to contain it. You can only open one solution at a time. [!INCLUDE[ssManStudioFull](../content/includes/ssManStudioFull_md.md)] does not allow you to create folders within projects. To organize your work, create multiple projects.  
  
To access Solution Explorer, click **SolutionÂ Explorer** on the **View** menu.  
  
### To create a new solution and project  
  
1.  On the **File** menu, point to **New**, and then click **Project**.  
  
2.  In the **New Project** dialog box, click a type of project.  
  
3.  Optionally modify the project name in the **Name** text box.  
  
4.  Optionally modify the path for the solution in the **Location** text box.  
  
5.  Optionally modify the solution name in the **Solution Name** text box.  
  
6.  Click **OK**.
