# isNamed Method (SQLServerSavepoint)
  Verifies if the savepoint has been named.  
  
## Syntax  
  
```  
  
public boolean isNamed()  
```  
  
## Return Value  
 **true** is the savepoint is named. Otherwise, **false**.
