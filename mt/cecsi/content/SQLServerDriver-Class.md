# SQLServerDriver Class
  Represents the [!INCLUDE[jdbcNoVersion](../content/includes/jdbcNoVersion_md.md)]. This class includes methods for connecting to a [!INCLUDE[ssNoVersion](../content/includes/ssNoVersion_md.md)] database, and for obtaining information about the JDBC driver.  
  
 **Package:** com.microsoft.sqlserver.jdbc  
  
 **Extends:** java.lang.Object  
  
 **Implements:** java.sql.Driver  
  
## Syntax  
  
```  
  
public final class SQLServerDriver  
```
