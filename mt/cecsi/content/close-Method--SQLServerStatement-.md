# close Method (SQLServerStatement)
  Releases this [SQLServerStatement](../content/SQLServerStatement-Class.md) object's database and JDBC resources immediately instead of waiting for them to be automatically released.  
  
## Syntax  
  
```  
  
public void close()  
```  
  
## Exceptions  
 [SQLServerException](../content/SQLServerException-Class.md)  
  
## Remarks  
 This close method is specified by the close method in the java.sql.Statement interface.
