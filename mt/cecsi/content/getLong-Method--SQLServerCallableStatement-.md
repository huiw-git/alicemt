# getLong Method (SQLServerCallableStatement)
  Retrieves the value of the designated parameter as a **long** in the Java programming language.  
  
## Overload List  
  
|Name|Description|  
|----------|-----------------|  
|[getLong (int)](../content/getLong-Method--int-.md)|Retrieves the value of the designated parameter as a **long** in the Java programming language given the parameter index.|  
|[getLong (java.lang.String)](../content/getLong-Method--java.lang.String-.md)|Retrieves the value of the designated parameter as a **long** in the Java programming language given the parameter name.|
