# Job Properties - New Job (Alerts Page)
Use this page to view and organize the alerts for a [!INCLUDE[msCoName](../content/includes/msCoName_md.md)] [!INCLUDE[ssNoVersion](../content/includes/ssNoVersion_md.md)] Agent job.  
  
## Options  
**Alert list**  
Lists the alerts for this job.  
  
**Add**  
Create a new alert for the job.  
  
**Edit**  
Modify the selected alert definition.  
  
**Remove**  
Remove the selected alert from the job, and delete the alert.
