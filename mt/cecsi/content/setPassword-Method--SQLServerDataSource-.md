# setPassword Method (SQLServerDataSource)
  Sets the password that will be used to connect to [!INCLUDE[ssNoVersion](../content/includes/ssNoVersion_md.md)].  
  
## Syntax  
  
```  
  
public void setPassword(java.lang.String password)  
```  
  
#### Parameters  
 *password*  
  
 A **String** that contains the password.
