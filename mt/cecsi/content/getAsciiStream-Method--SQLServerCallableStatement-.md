# getAsciiStream Method (SQLServerCallableStatement)
  Retrieves the value of the designated parameter as a stream of **ASCII** characters.  
  
## Overload List  
  
|Name|Description|  
|----------|-----------------|  
|[getAsciiStream &#40;int&#41;](../content/getAsciiStream--int-.md)|Retrieves the value of the designated parameter as a stream of **ASCII** characters given the parameter index.|  
|[getAsciiStream &#40;java.lang.String&#41;](../content/getAsciiStream--java.lang.String-.md)|Retrieves the value of the designated parameter as a stream of **ASCII** characters given the parameter name.|
