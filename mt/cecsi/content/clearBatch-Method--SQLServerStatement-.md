# clearBatch Method (SQLServerStatement)
  Empties the current list of SQL commands for this [SQLServerStatement](../content/SQLServerStatement-Class.md) object.  
  
## Syntax  
  
```  
  
public void clearBatch()  
```  
  
## Exceptions  
 [SQLServerException](../content/SQLServerException-Class.md)  
  
## Remarks  
 This clearBatch method is specified by the clearBatch method in the java.sql.Statement interface.
