# getCharacterStream Method (SQLServerCallableStatement)
  Retrieves the value of the designated parameter as a java.io.Reader object.  
  
## Overload List  
  
|Name|Description|  
|----------|-----------------|  
|[getCharacterStream &#40;int&#41;](../content/getCharacterStream--int-.md)|Retrieves the value of the designated parameter as a java.io.Reader object given the parameter index.|  
|[getCharacterStream &#40;java.lang.String&#41;](../content/getCharacterStream--java.lang.String-.md)|Retrieves the value of the designated parameter as a java.io.Reader object given the parameter name.|
