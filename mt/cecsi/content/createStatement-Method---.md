# createStatement Method ()
  Creates a [SQLServerStatement](../content/SQLServerStatement-Class.md) object for sending SQL statements to the database.  
  
## Syntax  
  
```  
  
public java.sql.Statement createStatement()  
```  
  
## Return Value  
 The Statement object.  
  
## Exceptions  
 [SQLServerException](../content/SQLServerException-Class.md)  
  
## Remarks  
 This createStatement method is specified by the createStatement method in the java.sql.Connection interface.
