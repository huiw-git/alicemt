# getDouble Method (SQLServerResultSet)
  Retrieves the value of the designated column in the current row of this [SQLServerResultSet](../content/SQLServerResultSet-Class.md) object as a **double** in the Java programming language.  
  
## Overload List  
  
|Name|Description|  
|----------|-----------------|  
|[getDouble \(int\)](../content/getDouble-Method--int---SQLServerResultSet-.md)|Retrieves the value of the designated column index in the current row of this [SQLServerResultSet](../content/SQLServerResultSet-Class.md) object as a **double** in the Java programming language.|  
|[getDouble \(java.lang.String\)](../content/getDouble-Method--java.lang.String---SQLServerResultSet-.md)|Retrieves the value of the designated column name in the current row of this [SQLServerResultSet](../content/SQLServerResultSet-Class.md) object as a **double** in the Java programming language.|
