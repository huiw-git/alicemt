# Proxy Account Properties (References Tab)
This read\-only page lists the job steps that reference a [!INCLUDE[msCoName](../content/includes/msCoName_md.md)] [!INCLUDE[ssNoVersion](../content/includes/ssNoVersion_md.md)] Agent proxy account.  
  
## Options  
**Job name**  
Names of jobs that use this proxy account.  
  
**Job step**  
Names of job steps in the job that use this proxy account  
  
**Subsystem**  
Subsystem this proxy account uses.
