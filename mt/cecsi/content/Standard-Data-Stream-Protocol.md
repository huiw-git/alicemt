# Standard Data Stream Protocol

  <introduction>
    <para>A standard data stream protocol is one way to access data in heterogeneous DBMSs. In fact, standard data stream protocols already exist: the ANSI/ISO Remote Database Access (RDA) standard (<externalLink><linkText>http://webstore.ansi.org/ansidocstore/product.asp?sku=ISO%2FIEC+9579%3A2000</linkText><linkUri>http://webstore.ansi.org/ansidocstore/product.asp?sku=ISO%2FIEC+9579%3A2000</linkUri></externalLink>) and the IBM Distributed Relational Database Architecture (DRDA) (<externalLink><linkText>http://www.netstoreusa.com/cbbooks/007/0070351384.shtml</linkText><linkUri>http://www.netstoreusa.com/cbbooks/007/0070351384.shtml</linkUri></externalLink>). Although both systems show promise, neither is widely implemented today.</para>
