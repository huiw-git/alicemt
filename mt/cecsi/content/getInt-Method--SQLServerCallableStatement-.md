# getInt Method (SQLServerCallableStatement)
  Retrieves the value of the designated parameter as an **int** in the Java programming language.  
  
## Overload List  
  
|Name|Description|  
|----------|-----------------|  
|[getInt \(int\)](../content/getInt-Method--int-.md)|Retrieves the value of the designated parameter as an **int** in the Java programming language given the parameter index.|  
|[getInt \(java.lang.String\)](../content/getInt-Method--java.lang.String-.md)|Retrieves the value of the designated parameter as an **int** in the Java programming language given the parameter name.|
