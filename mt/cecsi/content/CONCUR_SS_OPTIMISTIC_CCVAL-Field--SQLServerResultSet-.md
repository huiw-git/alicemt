# CONCUR_SS_OPTIMISTIC_CCVAL Field (SQLServerResultSet)
  Used to specify a [!INCLUDE[ssNoVersion](../content/includes/ssNoVersion_md.md)] read/write optimistic concurrency type with no row locks.  
  
## Syntax  
  
```  
  
public static final int CONCUR_SS_OPTIMISTIC_CCVAL  
```  
  
## Field Value  
 An **int** value of 1010.
