# CONCUR_SS_OPTIMISTIC_CC Field (SQLServerResultSet)
  Used to specify a [!INCLUDE[ssNoVersion](../content/includes/ssNoVersion_md.md)] read\/write optimistic concurrency type with no row locks.  
  
## Syntax  
  
```  
  
public static final int CONCUR_SS_OPTIMISTIC_CC  
```  
  
## Field Value  
 An **int** value of 1008.
