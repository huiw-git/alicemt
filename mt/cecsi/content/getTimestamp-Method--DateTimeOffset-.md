# getTimestamp Method (DateTimeOffset)
  Returns the point in time represented by this **DateTimeOffset** object as a java.sql.Timestamp.  
  
## Syntax  
  
```  
  
public java.sql.Timestamp getTimestamp()  
```  
  
## Return Value  
 A value of type java.sql.Timestamp.
