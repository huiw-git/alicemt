# getTimestamp Method (SQLServerCallableStatement)
  Retrieves the value of the designated parameter as a java.sql.Timestamp object in the Java programming language.  
  
## Overload List  
  
|Name|Description|  
|----------|-----------------|  
|[getTimestamp \(int\)](../content/getTimestamp-Method--int-.md)|Retrieves the value of the designated parameter as a java.sql.Timestamp object in the Java programming language given the parameter index.|  
|[getTimestamp \(int, java.util.Calendar\)](../content/getTimestamp-Method--int--java.util.Calendar-.md)|Retrieves the value of the designated parameter as a java.sql.Timestamp object in the Java programming language given the parameter index, using a Calendar object.|  
|[getTimestamp \(java.lang.String\)](../content/getTimestamp-Method--java.lang.String-.md)|Retrieves the value of the designated parameter as a java.sql.Timestamp object in the Java programming language given the parameter name.|  
|[getTimestamp \(java.lang.String, java.util.Calendar\)](../content/getTimestamp-Method--java.lang.String--java.util.Calendar-.md)|Retrieves the value of the designated parameter as a java.sql.Timestamp object in the Java programming language given the parameter name, using a Calendar object.|
