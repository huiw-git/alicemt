# getClob Method (SQLServerCallableStatement)
  Retrieves the value of the designated JDBC BLOB parameter as a Clob object in the Java programming language.  
  
## Overload List  
  
|Name|Description|  
|----------|-----------------|  
|[getClob \(int\)](../content/getClob-Method--int-.md)|Retrieves the value of the designated JDBC BLOB parameter as a Clob object in the Java programming language given the parameter index.|  
|[getClob \(java.lang.String\)](../content/getClob-Method--java.lang.String-.md)|Retrieves the value of the designated JDBC BLOB parameter as a Clob object in the Java programming language given the parameter name.|
