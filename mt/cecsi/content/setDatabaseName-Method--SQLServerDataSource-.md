# setDatabaseName Method (SQLServerDataSource)
  Sets the database name to connect to.  
  
## Syntax  
  
```  
  
public void setDatabaseName(java.lang.String databaseName)  
```  
  
#### Parameters  
 *databaseName*  
  
 A **String** that contains the database name.
