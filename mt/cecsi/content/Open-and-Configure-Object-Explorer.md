# Open and Configure Object Explorer
Open Object Explorer by using the **View** menu in [!INCLUDE[ssManStudioFull](../content/includes/ssManStudioFull_md.md)]. Configure Obect Explorer options by using **Options** on the **Tools** menu.  
  
## Viewing Object Explorer  
Object Explorer is visible in [!INCLUDE[ssManStudio](../content/includes/ssManStudio_md.md)] by default. If you cannot see Object Explorer, on the **View** menu, click **ObjectÂ Explorer**.  
  
## Configuring Object Explorer Options  
To configure the Object Explorer settings, on the **Tools** menu, click **Options**, and then click **SQL Server Object Explorer**.
