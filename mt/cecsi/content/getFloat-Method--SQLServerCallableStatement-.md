# getFloat Method (SQLServerCallableStatement)
  Retrieves the value of the designated parameter as a **float** in the Java programming language.  
  
## Overload List  
  
|Name|Description|  
|----------|-----------------|  
|[getFloat (int)](../content/getFloat-Method--int-.md)|Retrieves the value of the designated parameter as a **float** in the Java programming language given the parameter index.|  
|[getFloat (java.lang.String)](../content/getFloat-Method--java.lang.String-.md)|Retrieves the value of the designated parameter as a **float** in the Java programming language given the parameter name.|
