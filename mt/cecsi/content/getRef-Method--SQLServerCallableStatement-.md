# getRef Method (SQLServerCallableStatement)
  Retrieves the value of the designated parameter as a Ref object in the Java programming language.  
  
## Overload List  
  
|Name|Description|  
|----------|-----------------|  
|[getRef \(int\)](../content/getRef-Method--int-.md)|Retrieves the value of the designated parameter as a Ref object in the Java programming language given the parameter index.|  
|[getRef \(java.lang.String\)](../content/getRef-Method--java.lang.String-.md)|Retrieves the value of the designated parameter as a Ref object in the Java programming language given the parameter name.|
