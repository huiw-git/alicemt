# getApplicationIntent Method (SQLServerDataSource)
  Returns the value of the **applicationIntent** connection property.  
  
## Syntax  
  
```vb  
public java.lang.String getApplicationIntent();  
```  
  
## Return Value  
 Returns the value of the **applicationIntent** connection property.  
  
## Remarks  
 For more information about the **applicationIntent** connection property, see [Setting the Connection Properties](../content/Setting-the-Connection-Properties.md).
