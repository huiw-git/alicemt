    <para>The data types supported by DBMSs vary considerably. An application can determine the names and characteristics of supported data types by calling <legacyBold>SQLGetTypeInfo</legacyBold>. Because of wide variation in data type names, the application must use the data type names returned by <legacyBold>SQLGetTypeInfo</legacyBold> in <legacyBold>CREATE TABLE</legacyBold> statements. For more information, see <legacyLink xlink:href="7332d93e-44db-4132-9c10-988dbc13369e">Data Types in ODBC</legacyLink>.</para>
